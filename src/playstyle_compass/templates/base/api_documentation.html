API Documentation

Get List of Games

GET /api/games/

Description: Retrieves a paginated list of games.

Filters: Use query parameters to filter results.

  • platforms - Filter by platform (e.g., PC, PS5).
  • themes - Filter by game themes.
  • concepts - Filter by concepts.
  • genres - Filter by game genres.
  • release_date - Filter by release date.
  • developers - Filter by developers.
  • is_casual - Filter by casual games.
  • is_popular - Filter by popular games.
  • average_score - Filter by minimum score.
  • franchises - Filter by game franchises.

Ordering: You can order the games by the following fields:

  • title - Order by game title.
  • release_date - Order by release date.
  • average_score - Order by average score.

Pagination: Use the following parameters to control the pagination:

  • limit - Number of results per page (default: 100, max: 100).
  • offset - Skip the first N results (useful for pagination).

Example: /api/games/?platforms=PC&genres=Action&is_popular=true&ordering=release_date&limit=10&offset=0

Explanation: This request will retrieve a paginated list of popular action games available on PC, ordered by release date, with 10 results per page, starting from the first result.

Get Game Details

GET /api/games/<id>/

Description: Retrieves details of a specific game by ID.

Query Parameters:

  • guid - Unique global identifier.
  • title - Title of the game.
  • description - Detailed description.
  • overview - Short game overview.
  • genres - Game genres.
  • platforms - Platforms the game is available on.
  • themes - Game themes.
  • image - Main game image URL.
  • release_date - Release date of the game.
  • developers - Developers of the game.
  • game_images - Additional game images.
  • similar_games - Similar games.
  • dlcs - List of downloadable content (DLCs).
  • franchises - Game franchises.
  • videos - Links to game videos.
  • concepts - Game concepts.
  • is_casual - Boolean flag for casual games.
  • is_popular - Boolean flag for popular games.
  • playtime - Estimated playtime.
  • pc_req_min - Minimum PC requirements.
  • pc_req_rec - Recommended PC requirements.
  • mac_req_min - Minimum macOS requirements.
  • mac_req_rec - Recommended macOS requirements.
  • linux_req_min - Minimum Linux requirements.
  • linux_req_rec - Recommended Linux requirements.
  • average_score - Average game rating.
  • total_reviews - Total number of reviews.
  • translated_description_ro - Translated description (Romanian).
  • translated_overview_ro - Translated overview (Romanian).

Example: /api/games/12/?fields=title,genres,release_date
